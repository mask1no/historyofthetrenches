## History of the Trenches

Community-maintained crypto history archive and timeline. Built with Next.js (App Router), TypeScript, and Tailwind.

### Quickstart

```bash
pnpm install
pnpm dev
pnpm build && pnpm start
```

Open `http://localhost:3000`.
